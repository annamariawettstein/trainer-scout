## ğŸ“Š Methodology

### Score Calculation
Computes composite score from 7 metrics:
- 1.3: Teaching style / concentration
- 1.4: Opportunity to participate
- 2.8: Theory into practice
- 2.9: Flexibility and adaptation
- v1_1.2: Trainer concentration
- v2_1.1: Trainer motivation
- v2_1.2: Explanation clarity

### Improvement Analysis
For each trainer:
1. Sort feedback chronologically
2. Split into early and late halves
3. Calculate: `improvement = mean(late) - mean(early)`
4. Rank by improvement score

### Quote Extraction
Scans text feedback for positive keywords:
- excellent, great, helpful, engaging
- patient, clear, friendly, flexible
- motivated, fun, approachable
